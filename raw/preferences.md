# 最新進度（2026-06-23）

## 已完成功能

### 一鍵分享報獎分享圖
- `ShareRewardCard.ts` — 掛在 Prefab 節點上，自動建 Camera + RenderTexture 截圖，不需手動設 Camera
- Inspector 拖入：amountLabel、multiplierLabel、gameIconSprite、vendorLogoSprite、qrCodeSprite
- 流程：填資料 → 等幀渲染 → Camera 截圖 → 輸出 File → 系統分享
- 整合到 `HighlightPanel.onCellShareClick` 內建，點 Share 直接走分享圖流程

### BitmapFont
- `assets/Highlight/Font/sharewin_num.fnt` — 紅金色數字（0-9, 逗號, 小數點），size=304
- `assets/Highlight/Font/sharewin_numblue.fnt` — 藍色數字+x+X，size=196

### 多語系
- `assets/Highlight/MultiLang/` — 7 種語系 JSON（zh-tw, zh-cn, en-us, th-th, vi-vn, ms-my, es-es）
- CSV 第一欄是 Key，跑 `python convert.py` 自動產出 JSON
- 使用方式：Label 掛 `UIMultiLang` 組件，key 欄位填 key 名稱

### Info 彈窗
- `HighlightPanel` 新增 `infoBtn` / `infoPopup` property
- 點 InfoButton 開啟彈窗，點畫面任意位置關閉
- 彈窗 Label 用 `UIMultiLang` key=`HIGHLIGHT_INFO`

### HighlightAutoLoader（測試用）
- 模擬正式流程：建 MockHighlightSystem + HighlightModuleIntegration
- 攔截 SendGetReplay / SetFavorite，回假資料
- 所有按鈕走 Integration 完整鏈路
